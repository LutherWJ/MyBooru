# MyBooru Keyboard Navigation
MyBooru aims to allow for full keyboard navigation for the most efficient user experience.

## Global Navigation
- CTRL+T: Open new tab with prompt to select base page
- CTRL+W: Close currently selected tab
- CTRL+TAB: Move to next tab index
- CTRL+SHIFT+TAB: Move to previous tab index
- CTRL+[Num 0-9]: Immediately jump to tab at selected index (0 is interpreted as 10)

## Gallery Navigation
- W/A/S/D: Navigate selecting images
- SPACEBAR: Open Selected image
- Q: Select the search bar to begin typing
- Enter: Submit search
- Z: Navigate to previous page (Search results are paginated)
- X: Navigate to next page
- H: Select leftmost image on current row
- J: Select top image on current column
- K: Select bottom image on current column
- L: Select rightmost image on current row

## File Upload Navigation
- W/A/S/D: Select Different Elements
- SPACEBAR: Left click/activate selected element
- Q: Select tag box
